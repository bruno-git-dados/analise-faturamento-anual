# Análise de Desempenho Escolar (SAT) com Python

Este projeto realiza uma análise estatística e visual sobre o desempenho de 15 escolas brasileiras fictícias (mas que utilizam a distribuição estatística de dados de Nova York). 

O objetivo principal foi responder a três perguntas de negócio:
1. Quais escolas possuem os melhores resultados em matemática?
2. Quais são as 10 escolas com melhor desempenho geral (soma de todas as matérias)?
3. Qual região apresenta a maior desigualdade (desvio padrão) nas notas?

## 🛠️ Tecnologias Utilizadas
- Python 3
- Pandas: Para a manipulação, agrupamento (`groupby`) e tratamento estatístico dos dados.
- Seaborn e Matplotlib: Para a geração de gráficos de barras horizontais independentes.

## 📊 Resultados Visuais
O script gera duas telas gráficas automatizadas:
- O ranking das 10 melhores instituições de ensino.
- O mapeamento do Top 5 em matemática.
